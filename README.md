# Job Applicants Monthly Analysis

This project analyzes job applicants data over one year
to identify monthly patterns in recruitment activity.

## Dataset
- Columns: ID, Date, Job Title, Job Location, Hired, Easy Apply, Month
- Records: 3000+

## Analysis
- Monthly count of applicants
- Average, maximum and minimum values by month and total number of applications

## Key Insights
- Applicant volume varies significantly by month
- Clear peak and low seasons are observed

## Limitations
- Descriptive analysis only
